# 目录

本仓库为某9人小组维护的前端面试题库，包含不限于Vue面试题，React面试题，JS面试题，HTTP面试题，工程化面试题，CSS面试题，算法面试题，大厂面试题，高频面试题

## 🤡 这个仓库主要内容是?

仓库目前主题以前端面试题为主，后续可能会扩展更多的分支

在这个信息爆炸，前端生态百花齐放的时代，我们每找寻一个答案都要换一种形式，这简直太奢侈了

我们的目标是做最适合自己的前端面试仓库，用心收录各种面试题，高频面试题，知识点面试题，用心做好每一道题值得参考的题解

## 🤡 你们的本质就是面试题吗？

不是，一直在强调自身的努力，这个仓库也仅且是一个附属品

我们只是以面试题的形式作为切入点，根据面试题去反推学习更具体的知识点，绝不仅限于应对面试，要真真切切在学习过程中得到蜕变

它的背后是思考，总结，更是坚持

在过程中看清自已，在结果中遇见自己，我们都值得有更美好的未来
