# 10. 什么是SPA

## 逐字稿

SPA 称为单页应用程序, 也就是整个网站只有一个 html 页面. 它的优点是相对于传统混合式开发, 减少了请求体积, 加快页面响应速度, 降低了对服务器的压力. 缺点是因为使用了 ajax, 导致页面不利于 SEO.

我们所熟知的 JS 框架 react, vue, angula r都属于 SPA
